# Workshop Portal

## ⏳ Step 1: Wait for Installation
Your computer is currently installing **Node.js**. This is a one-time setup that is currently running in your background terminal. 
**Wait for this to finish** before proceeding.

## 🚀 Step 2: Setup
Once the installation above is done, open a **new terminal** in this folder and run:
```bash
./setup.sh
```
This will install all the project libraries.

## 🏃 Step 3: Run
You need **two** terminal windows open to run the app:

**Terminal 1 (Server):**
```bash
cd server
npm run dev
```

**Terminal 2 (HyperClass Frontend):**
```bash
cd client
npm run dev
```

**Terminal 3 (Adilitix Frontend):**
```bash
cd adilitix
npm run dev
```

Then open your browser:
- **HyperClass:** Usually `http://localhost:5173`
- **Adilitix:** Usually `http://localhost:5174`
